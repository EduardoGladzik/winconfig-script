Description
=============================
PowerShell script designed to automate tasks of my daily work routine.
The script is designed to be launched from a pen-drive.

Functionalities
=============================
- Report: generates a report with relevant information about the computer, such as name and printer settings.
- Apps: Install apps via Chocolatey package manager.
- Winconfig: configure settings like neetwork sharing and sleep time.

Broken
=============================
sometimes app instalation can broke because of how updates are made in Chocolatey. It takes up to 6h for them to fix this kind of issue..

- Windows performance settings
- Enable Telnet and SMBV1.0 clients
